# AI Studio Pro Enterprise

## Coding Rules

1. Every module must have a docstring.

2. Every class must have a docstring.

3. Every function must have a docstring.

4. Use descriptive variable names.

5. Never duplicate code.

6. Test before commit.

7. Push every completed feature.

8. One feature = One Git commit.

9. Follow Enterprise Architecture.

10. Keep the project production ready.
