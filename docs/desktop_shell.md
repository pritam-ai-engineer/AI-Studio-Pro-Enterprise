## Version

v0.2.0-beta

## Goal

Provide the enterprise desktop framework for AI Studio Pro Enterprise.

---

## Components

- Main Window
- Menu Bar
- Toolbar
- Sidebar
- Workspace
- Dashboard
- Status Bar
- Theme Manager
- Navigation Controller

---

## Responsibilities

### Main Window

Owns the application window.

Does NOT contain business logic.

---

### Menu Bar

Application menus.

---

### Toolbar

Quick actions.

---

### Sidebar

Navigation only.

---

### Workspace

Displays pages.

---

### Dashboard

Default landing page.

---

### Status Bar

Application status.

---

### Theme Manager

Application appearance.

---

### Navigation Controller

Controls page switching.
